Kolor tła

 Dzięki temu poleceniu można ustalić kolor tła dla danego elementu.
Składnia reguły:

selektor {background-color: kolor;}

p#first { background-color: red; }

Tło Obrazkowe

 Dzięki temu poleceniu można ustalić tło obrazkowe dla danego elementu.
Składnia reguły:

selektor {background-image: url(adres);}

p#first { background-image: url("file:///c:/CSS/_rozwiazanie/tla_marginesy/grafiki/tlo_1.jpg"); }

Powtarzanie tła

 Dzięki temu poleceniu można ustalić jeden z czterech sposobów powtarzania się obrazka jako tła.
Składnia reguły:

selektor {background-repeat: powtarzanie;}

p#first { background-repeat: repeat; }

Pozycja tła

 Dzięki temu poleceniu można ustalić pozycję obrazka w ramach tła. Niemożliwe to jest przy wartości atrybutu powtarzania: repeat.
Składnia reguły:

selektor {background-position: pozycja;}

p#first { background-position: center; }

Zaczepienie tła

 Poleceniem tym można sprawić, że tło będzie nieruchome w czasie przewijania zawartości strony.
Składnia reguły:

selektor {background-attachment: zaczepienie;}

p#first { background-attachment: scroll; }

Łączenie atrybutów tła

 Wyraz selektor ma zastosowanie do wszystkich elementów.
Składnia reguły:

selektor {background: wartości atrybutów;}

p#first { color: #800000; background: #00ff00 url(../../grafiki/cwiczenie_13/tlo_css.jpg) norepeat scroll center }

Gradient liniowy

 Gradient to wypełnienie tłem płynnym przejściu kilku kolorów
Składnia reguły:

selektor { background: linear-gradient(kierunek, kolor1 odległość1, kolor2 odległość2...) }

p#first { background: linear-gradient(to bottom, black 0%, white 100%) }

Gradient promienisty

 Gradient promienisty to tło o płynnym przejściem kilku kolorów, wywołując efekt trójwymiarowej kuli lub oświetlenia latarki
Składnia reguły:

selektor { background: radial-gradient(kształt rozmiar at pozycja, kolor1 odległość1, kolor2 odległość2...) }

p#first { repeating-radial-gradient(white 33%, black 66%) }
